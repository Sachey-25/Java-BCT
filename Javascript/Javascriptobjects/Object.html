This operator in Javascript.

the this keyword refers to an object which object depends on how 'this' is being invoked.(used/called). The 'this' keyword refers to different depenging object logic.

the 'this' operator refers to objects depending on how it is used. In an object method, "this" refers to the object. Alone, "this" refers to the global object. In a function, "this" refers to the global object. In a function,in strict mode, "this" is undefined. In an event, this refers to the element that received the event. Methods like call(), apply() and bind() can refer to "this" to any object.
